***Chris's Scripts*** 

A constantly updating set of scripts used when running Linux-based computers both small and large. I created this repository because I was tired of prompting the AI to run the same set of commands over and over again, wasting valuable tokens and time. So I thought, why not use the machine to build the machines (deterministic scripts)? 

What I currently have

- Ethernet 
- Wi-Fi (depending on whether your computer has a Wi-Fi module or not) 
- Tailscale 
- Speedtest
- SSH
- Screen (to be able to connect to an external PC, depends on if your distro supports it) 
- Bluetooth (depending on whether you can support Bluetooth or not) 
- Git (including creating keys for your GitHub) 
- Getting the ACM values and IDs of USB connections 
- Getting the speeds of said USB connections
- Running kernel debugging
- Getting disk and RAM info 
- Repartitioning an SSD 

... with many more to come. 



Also includes skills for your agents!
